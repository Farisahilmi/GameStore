# Usulan Peningkatan UI/UX "Premium & Immersive"

Untuk membuat GameStore terasa lebih premium dan modern (mirip Steam/Epic Games baru), berikut adalah rencana peningkatannya:

## 1. Immersive Navigation & Search (UX)
*   **Search Autocomplete**: Menambahkan fitur pencarian cerdas di [Navbar.jsx](file:///d:/GameStore/client/src/components/Navbar.jsx). Saat user mengetik, akan muncul dropdown berisi hasil game yang cocok secara real-time (lengkap dengan gambar kecil dan harga).
*   **Contextual Top Bar**: Menambahkan bar tipis di atas navbar yang menampilkan informasi saldo Wallet atau event Sale yang sedang berlangsung secara dinamis.

## 2. Immersive Game Exploration (UI)
*   **3D Hover Effects**: Menambahkan efek tilt 3D pada [GameCard.jsx](file:///d:/GameStore/client/src/components/GameCard.jsx) menggunakan `framer-motion`. Ini memberikan kesan "kedalaman" saat kursor bergerak di atas kartu game.
*   **Parallax Header**: Di [GameDetails.jsx](file:///d:/GameStore/client/src/components/GameDetails.jsx), kita akan membuat header gambar game bergerak sedikit saat di-scroll (efek parallax) untuk memberikan kesan sinematik.

## 3. Interactive Feedback (UX)
*   **Page Transitions**: Menggunakan `AnimatePresence` di [App.jsx](file:///d:/GameStore/client/src/App.jsx) agar perpindahan antar halaman terasa mulus (fade/slide) daripada langsung "pindah" kasar.
*   **Micro-animations**: Menambahkan animasi "Pop" kecil saat klik tombol "Add to Cart" atau "Wishlist" untuk memberikan feedback yang memuaskan bagi user.

## 4. Design System Refinement (UI)
*   **Glassmorphism Everywhere**: Menyatukan gaya "Glassmorphism" (efek kaca transparan) pada semua Modal dan Card agar konsisten di seluruh aplikasi.
*   **Better Skeleton Loaders**: Memperhalus tampilan loading dengan gradien yang bergerak (shimmer effect) agar user tidak merasa aplikasi "hang" saat menunggu data.

Untuk membuat GameStore terasa seperti aplikasi "Real World" yang hidup dan interaktif, kita tidak bisa hanya mengandalkan fitur dasar. Berikut adalah 3 fitur prioritas yang akan mengubah website ini menjadi platform komunitas yang sesungguhnya:

## 1. Sistem Chat Persisten (Database-Backed Chat) ğŸ’¬
Saat ini, fitur chat kita masih bersifat "sementara" (hilang saat refresh). Aplikasi nyata harus menyimpan riwayat percakapan.
- **Tujuan:** Agar user bisa melihat chat lama, melanjutkan obrolan kemarin, dan tidak kehilangan pesan saat koneksi putus.
- **Implementasi:**
  - Membuat model database `Message` (sender, receiver, content, timestamp).
  - API untuk memuat riwayat chat saat jendela chat dibuka.
  - Menandai pesan "Sudah Dibaca" (Read receipts).

## 2. Sistem Review & Rating Game â­
Toko game tanpa review terasa "kosong". Kita butuh fitur di mana user bisa menilai game yang sudah mereka beli.
- **Tujuan:** Memberikan validasi sosial (Social Proof) bahwa game tersebut bagus, dan memicu interaksi antar user.
- **Implementasi:**
  - User hanya bisa mereview game yang sudah dibeli (di Library).
  - Rating Bintang (1-5) + Komentar Teks.
  - Tampilan rata-rata rating di halaman game.

## 3. Activity Feed (Kabar Teman) ğŸ“¢
Fitur ala sosial media (seperti Steam Activity) untuk melihat apa yang sedang dilakukan teman.
- **Tujuan:** Supaya user tidak merasa sendirian. Mereka bisa melihat aktivitas teman secara real-time.
- **Implementasi:**
  - Otomatis mencatat event: "Budi baru saja membeli GTA V", "Siti memberikan review 5 bintang untuk Elden Ring", "Joko menjadi teman baru Anto".
  - Tampilan Feed di Dashboard/Home.

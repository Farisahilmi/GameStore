# Rencana Implementasi Fitur & UI Premium

Untuk membuat GameStore semakin premium dan interaktif, saya akan mengimplementasikan 3 fitur visual utama berikut:

## 1. Dynamic Video Hero (Immersive)
Mengganti gambar statis di halaman utama dengan **Video Trailer** yang berputar otomatis. Ini akan memberikan kesan "hidup" dan sinematik seperti Steam/Epic Store saat ada event besar.
*   **Komponen**: `Hero.jsx`
*   **Implementasi**:
    *   Menggunakan tag `<video>` dengan properti `autoPlay`, `muted`, `loop`.
    *   Menambahkan overlay gradient agar teks tetap terbaca di atas video.
    *   Menambahkan tombol "Play with Sound" opsional.

## 2. Skeleton Loading Screens (Perceived Performance)
Saat ini user melihat loading spinner biasa atau layar kosong saat data sedang diambil. Kita akan menggantinya dengan animasi **"Shimmering Skeleton"** yang meniru bentuk kartu game.
*   **Komponen**: `Skeleton.jsx` (Baru) & Integrasi ke `BrowseGames.jsx`, `Hero.jsx`.
*   **Benefit**: User merasa website lebih cepat karena layout sudah terlihat instan.

## 3. Interactive "Like" Animation (Micro-interaction)
Memberikan feedback visual yang memuaskan saat user menambahkan game ke wishlist.
*   **Komponen**: `GameDetails.jsx`
*   **Implementasi**:
    *   Efek tombol "meledak" atau berdenyut saat diklik.
    *   Menggunakan `framer-motion` untuk animasi scale dan color transition.

---

## Langkah Eksekusi:
1.  **Update `Hero.jsx`**: Tambahkan logika untuk memutar video background jika tersedia (kita akan gunakan video sample dari internet untuk demo).
2.  **Buat `Skeleton.jsx`**: Buat komponen skeleton yang cantik dengan animasi gradien bergerak.
3.  **Integrasi Skeleton**: Pasang skeleton di `BrowseGames.jsx` saat state `loading` aktif.

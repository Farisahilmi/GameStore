# Rencana Implementasi UI/UX Super Polish & Theme Adaptive

## 1. Fondasi Tema (ThemeContext.jsx)
- Menambahkan properti desain baru: `shadows`, `glass`, `ring`, `buttonPrimary`, `input`.
- Contoh: Tema Cyberpunk akan mendapatkan aksen `shadow-[0_0_15px_rgba(6,182,212,0.5)]`.

## 2. Global Styling (index.css)
- Implementasi CSS Variables untuk warna primer/sekunder.
- Custom scrollbar yang warnanya mengikuti variabel CSS tersebut.
- Penambahan animasi `.animate-float` untuk elemen dekoratif.

## 3. Peningkatan Navbar (Navbar.jsx)
- Desain ulang state 'scrolled' dengan border gradien tipis.
- Animasi dropdown yang lebih elastis menggunakan framer-motion.

## 4. High-End Cards & Hero
- Penambahan efek 'Spotlight' pada GameCard (cahaya yang mengikuti kursor).
- Layout Hero yang lebih lega dengan tipografi yang lebih berani.

## 5. Micro-Interactions
- Haptic feedback visual (skala kecil) pada semua elemen klik.
- Loading state yang lebih artistik (tidak hanya spinner).

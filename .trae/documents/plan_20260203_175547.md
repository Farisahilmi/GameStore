## Rencana Pengembangan Fitur GameStore

### 1. Peningkatan Form Tambah/Edit Game (ManageGames.jsx)
*   **System Requirements**: Menambahkan field input untuk spesifikasi Minimum dan Recommended (OS, CPU, RAM, GPU, Storage).
*   **Game Metadata**: Menambahkan field untuk link sosial (Discord, Website) dan status "Early Access".
*   **Content Rating**: Menambahkan pilihan rating umur (General, Teen, Mature).

### 2. Integrasi Notifikasi & Sosial
*   **Broadcast Update**: Otomatis mengirim notifikasi ke semua follower saat publisher memposting "Patch Note" atau merilis game baru.
*   **Voucher Management**: Membuat UI sederhana bagi Publisher untuk membuat kode voucher diskon khusus game mereka.

### 3. Dashboard Analitik Ringkas
*   Menambahkan mini-chart atau statistik cepat di setiap item game pada daftar `ManageGames.jsx` agar publisher bisa langsung melihat performa game tersebut tanpa harus ke halaman analitik utama.

### Langkah Selanjutnya:
1. Memperbarui `schema.prisma` untuk mendukung field baru (Requirements, Social Links).
2. Memperbarui `gameService.js` dan `gameController.js` di sisi backend.
3. Merombak UI `ManageGames.jsx` menjadi sistem "Step Wizard" atau tabbed form agar tidak terlalu panjang.

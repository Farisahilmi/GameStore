## Pilihan Fitur Baru:

### 1. Social & Community Layer
- **Fitur:** Friend System & Activity Feed.
- **Manfaat:** Membuat user betah berlama-lama di website karena ada interaksi sosial.
- **Langkah:** Membuat sistem add friend, halaman profil user, dan feed aktivitas di Home.

### 2. Game Update Logs (Patch Notes)
- **Fitur:** Tab Update resmi di halaman detail game.
- **Manfaat:** User tahu bahwa game tersebut terus dikembangkan (meningkatkan kepercayaan pembeli).
- **Langkah:** Menambahkan fitur bagi Publisher untuk memposting update dan menampilkannya di Game Details.

### 3. Discovery Queue
- **Fitur:** Rekomendasi game harian yang dipersonalisasi.
- **Manfaat:** Membantu user menemukan game baru dengan cara yang menyenangkan.
- **Langkah:** Membuat algoritma sederhana untuk menyajikan game yang belum dimiliki user secara bergantian.

### 4. Gifting System
- **Fitur:** Beli game untuk teman.
- **Manfaat:** Meningkatkan potensi penjualan dan interaksi antar user.
- **Langkah:** Modifikasi sistem Checkout agar bisa memilih penerima dari daftar teman.

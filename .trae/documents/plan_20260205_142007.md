# Rencana Testing & Preview Website

Untuk memastikan website berjalan dengan semestinya, saya akan mengaktifkan seluruh sistem (Full Stack) agar Anda bisa mencobanya langsung.

## Langkah-langkah:

1.  **Nyalakan Backend Server (Port 3000)**
    *   Ini akan mengaktifkan API, koneksi database, dan fitur real-time (Socket.io).
    *   Saya akan memantau log untuk memastikan tidak ada error database.

2.  **Nyalakan Frontend Client (Port 5173)**
    *   Ini akan menjalankan tampilan antarmuka React.

3.  **Verifikasi Koneksi**
    *   Memastikan frontend bisa bicara dengan backend (tidak ada error `Network Error` atau `Connection Refused`).
